# Guide til webutvikling

## Hvem, hva og hvorfor?
- Kvem er nettsida for = Målgruppen vår
  - Personleg: for oss, hobby, delt til venner og familie, private prosjekt
  - Offentlig: for andre, delt til arbeidsgiver/kursholder, seriøs tone

  - Aldergrupper: Har eller ikkje har teknologisk bakgrunn
    - Mørk eller lys nettsde
    - Antall skjulte deler av siden (Meny)
    - Tilgjengelighet, synlighet og bruksvennlighet på sida

- Kva skal nettsida inneholde = Innhold og funksjonalitet
  - Tekst vs bilder
  - Minimal mobil versjon -> Maximal desktop versjon
  - Personlig gjøre

- Kvifor eksisterer nettsida = Målet til sida
  - Fremvisning: portfolio, bildegalleri, etc
  - Informative: Artikler, encyclopedia, aviser
  - kjøp og salg: Nettbutikk,

## header-elementet

- Logo
- Søkemotor
- Logg inn / Register
- Navigasjon

## main-elementet:

- Hero med viktigest info
  - Info om hva siden er
  - Spesielle tilbud
  - Nytt som skjer
- Tilegg seksjoner med mer info
  - Kort med informasjon
  - Dypere navigasjon
  - Reklame for siden/produkt

## footer-elementet:

- Logo
- Navigasjon
- Personvernserklæring / TOS
- SoMe
- Kreditasjon
